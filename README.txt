WC Missing Stocks plugin creates a report of missing stocks.

== Description ==

WC Missing Stocks is just one click install plugin that creates a report of missing stocks, which makes it easy for shop owners to track all their misssing stocks.

The plugin list all products and its variations as well, so you can track not just the simple products but variant products as well.

= Features of the plugin includes =

*   Woocommerce Missing Stocks Report
*   Simple and Variant Products in Report
*   Search Filter
*   Date Range Filter
*   One click Install
*   Works as Adons to Woocommerce Reports


= How to use =

1. First installed and active the plugin
2. Once Activated you can find a new tab in the Woocommerce->Reports->Stock->Missing Stock
3. All missing stocks are listed for simple products and variant products
4. Search can filter records
5. Date Range can also filter by date

== Installation ==

### Uploading in WordPress Dashboard

1. First download the plugin ZIP file from wordpress.org
2. Log in to your website administrator panel
3. Go to the 'Add New' in the plugins dashboard, click “Upload Plugin”
4. Upload WC Missing Stocks ZIP file by choosing it from your download location
5. Click Install Now button
6. Then click Activate the Plugin button.
7. You can see the plugin installed on Wordpress left menu.

### Using The WordPress Plugin Dashboard

1. Go to the 'Add New' in the plugins dashboard
2. Search for 'wc-missing-stocks' or 'WC Missing Stocks'
3. Click Install Now button
4. Then click Activate Plugin button
5. You can see the plugin installed will add a new tab to Woocommerce->Reports->Stock->Missing Stock.
